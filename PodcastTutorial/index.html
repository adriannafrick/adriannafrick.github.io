Step 3:

Project manager opens in Audacity and Saves Project to Original folder.

Recommended: Save any changes or updates to Working folder. This way, if there are any problems, you can revert back to original.

Step 4:

Sync-Lock Tracks and use a Label Track to mark sections for editing/moving/etc.

Mark a section of audio by highlighting it. Hit Ctrl and B at same time (on PC) or ⌘-B on Mac to create a label.

Step 5:

Listen to podcast once through to find retakes and major edits. (Can be at “chipmunk” speed if needed.

Step 6:

Listen through closely for ums, uhs, and breaths. Highlight and delete these sections, as well as any lengthy pauses.

Step 7:

Run a High Pass filter (generally 110 Hz, 6db rolloff), a Low Pass Filter (generally 5000 Hz, 6db rolloff), and Normalize ( to make sure loud spikes or changes in volume don’t deafen your listeners. Listen to the podcast in headphones to make sure no one sounds like they are in a tin can or on the other side of a giant ballroom. Save project to a new file with the word “Mix” added to the end of it.

Note: It is easier to change edits before running any filters/normalizing. If you have to go back in and fix something in the file, it is easier to do it, then REmix the podcast. Also, if you mangle the mix process, you still have the working file with edits to fall back on.
